# homebrew-tap

Homebrew tap for [naonya3](https://github.com/naonya3)'s CLI tools.

## Install

```bash
brew install naonya3/tap/<formula>
```

## Available Formulae

### tka — Workflow declaration DSL for AI agents

```bash
brew install naonya3/tap/tka
```

[github.com/naonya3/tka](https://github.com/naonya3/tka)

## Adding a new tap

```bash
brew tap naonya3/tap
brew install <formula>
```
